API Verbs App

This small app will make one of the 4 basic HTTP requests to the HTTBin API: https://httpbin.org/

The goal is to enter students data in the format:
{
name: Student Name,
age: Integer Representing Age,
class: Random 4 digit Number (Optional parameter, it can be calculated)
}
When you use POST requests, enter the data in the same format and you should see which data was sent.
